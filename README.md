Meri-Awaaz-News-Portal 🚀

# 📢 Overview

Meri Awaaz is a next-generation news and media platform inspired by the experience of platforms like Instagram + YouTube, combining short videos, news sharing, social interaction, moderation, and AI-powered security monitoring.

The platform supports multiple user roles including:

👤 Guest Users
🧑 Registered Users
🛡️ Moderators
👑 Super Admin

It provides real-time notifications, secure content management, scalable video processing, and advanced moderation workflows.

✨ Key Features
🎥 Video & Media System
Instagram/Reels + YouTube-style video experience
Like / Dislike videos
Follow / Unfollow creators
Video reporting system
Chunk-based video streaming for faster playback
Reduced buffering using optimized video delivery
Telegram-based storage system
Telegram bots used as background workers for video processing and fetching
🔔 Real-Time Notification System

Users receive real-time updates for:

Profile views
New followers
Video interactions
Reports and moderation actions
Account activities

Example:

When someone views your profile, you receive a notification:
"User X viewed your profile"

👥 User Privacy System

Privacy-focused profile management:

Guest users cannot view user profiles
Only authenticated users can access profiles
Profile visibility controls
User activity protection
🏛️ Role Based Access Control (RBAC)
👑 Super Admin

Full platform control:

Manage all users
Promote users to moderators
Remove moderator privileges
Ban / Unban users
Delete accounts
Manage upload restrictions
Increase/decrease user limits
Review moderation decisions
Monitor platform security
🛡️ Moderator

Limited permissions:

Manage news/content only from assigned areas
Approve or reject videos
Review user reports
Moderate regional content

Moderators cannot access areas outside their permission scope.

🚨 Advanced Moderation & Security Flag System

Automated moderation monitoring:

Tracks moderator actions
Detects suspicious approvals/rejections
Escalates questionable decisions to Super Admin

Example:

If a moderator approves or rejects content without valid reports/reasons:

➡️ System automatically flags the action
➡️ Super Admin receives review request
➡️ Moderator receives a strike if the decision was incorrect

⚠️ Moderator Reputation System

Moderator strikes are:

Recorded permanently
Visible on moderator profiles
Used for accountability tracking

This creates transparent moderation management.

🤖 AI Blue Team Security System

Meri Awaaz supports integration with custom AI security agents.

The AI security layer monitors:

API activity
User behavior
Security risks
Suspicious requests
Abuse patterns

Capabilities:

Temporary user bans
Access restrictions
IP blocking
Risk-based actions
Security alerts
📊 Admin Security Dashboard

Real-time monitoring dashboard including:

Security events
User risk scores
Suspicious activity tracking
AI security decisions
Platform health monitoring
🔐 API Security

Implemented security protections:

Rate limiting on public endpoints
Abuse prevention
Access control
Authentication protection
Permission-based API access
